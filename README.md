# jupyterlab-broccoli-turtle

* Simple turtle blocks for jupyterlab-broccoli
* Attention: Almost impractical with FireFox because of FOUC. **Do not use with FireFox.**


## Requirement
* Pillow
  * pip install Pillow
* jbturtle >= 0.2.0
  * pip install jbturtle
